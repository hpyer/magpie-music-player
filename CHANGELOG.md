# 变动历史

本文件记录面向用户和插件开发者的重要变更。每次发布新版本前，应根据 git 历史整理对应版本的变动内容。

格式约定：

- 桌面应用版本使用 `## v1.2.0 - YYYY-MM-DD`
- npm 包版本也记录在本文件中，并在标题中标明包名
- GitHub Release 只会读取桌面应用 tag 对应的应用版本段落，例如 `v1.2.0` 读取 `## v1.2.0 - ...`

## v1.0.0 - 2026-07-21

### 新增

- 初始发布 Magpie Music Player 桌面应用。
- 支持本地音乐目录扫描、播放列表、播放控制和基础设置。
- 支持插件安装、配置、启用、禁用和删除。
- 支持插件黑名单拉取、签名校验、本地缓存和阻止提示。
- 提供 `@magpie-music/plugin-types` 插件类型包。
- 提供 `@magpie-music/plugin-cli` 插件创建、测试和打包命令。

### 说明

- Gitee 仓库仅作为镜像，正式 Release 以 GitHub 为准。
- 核心播放器应用使用 AGPLv3 许可。
- 插件类型包、插件 CLI、官方开源插件和模板使用 MIT 许可。
